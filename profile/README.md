## ✨ 私たちについて

**Team Rapportia** は、地域に根ざした個人店・こだわりのお店（カフェ、美容室、自然食品店、ギャラリーなど）のために、お店の**「想い」**が伝わる Web サイトの制作と、公開後のサポートを行っています。
