## ✨ 私たちについて

<br>

**Team Rapportia** は、地域に根ざした個人店・こだわりのお店のために、お店の **「想い」** が伝わる Web サイトの制作と、公開後のサポートを行っています。

<br>

### 💬 想いを形に

ヒアリングを通じて、お店の「らしさ」を丁寧に掘り起こし、言葉とデザインに落とし込みます。

<br>

### 🌱 育てるサイト

WordPress で、公開後もご自身で更新できるサイトを。専門知識がなくても安心です。

<br>

### 🤝 長期的な寄り添い

無料修正サポート・12ヶ月品質保証。公開後もずっとサポートし続けます。

<br>

「素敵なお店なのに、なかなか新しいお客様に知ってもらえない」  

「ホームページを作りたいけど、難しそうで踏み出せない」  

—— そんな声に応え、Web の専門知識がなくても、一緒にお店の魅力を整理して届けるお手伝いをしています。
