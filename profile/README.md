# Team Rapportia

**地域のお店と、未来をつなぐ。**

Team Rapportia は、地域に根ざした個人店・こだわりのお店のために、お店の「想い」が伝わる Web サイトの制作と、公開後のサポートを行っています。

- お店の「らしさ」をヒアリングし、言葉とデザインに落とし込んだサイトづくり
- 公開後もご自身で更新できる「育てるサイト」の提供
- 無料修正サポート・品質保証に代表される、長期的な寄り添い

「素敵なお店なのに、なかなか新しいお客様に知ってもらえない」「ホームページを作りたいけど、難しそうで踏み出せない」—— そんな声に応え、Web の専門知識がなくても、一緒にお店の魅力を整理して届けるお手伝いをしています。

---

この **Team Rapportia** オーガナイゼーションは、上記の活動を支える**開発用の GitHub 組織**です。公式サイトや関連するプロダクトのソースコード・開発作業は、この組織配下のリポジトリで管理されています。
